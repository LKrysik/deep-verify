## Status Colors

- **GREEN**: Portfolio healthy, all metrics within thresholds
- **YELLOW**: Some metrics approaching thresholds, attention needed
- **RED**: Critical thresholds breached OR any unmitigated NON_ERGODIC/FAT_TAIL risks

## Update Frequency

- **Real-time**: Active alerts, accumulation watches
- **Weekly**: Top risks, coverage metrics
- **Monthly**: Full portfolio review, strategy distribution
- **Quarterly**: Deep assessment refresh
