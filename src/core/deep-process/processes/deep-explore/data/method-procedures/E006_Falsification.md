# E006: Falsification

## Purpose

Test beliefs by trying to disprove them, not confirm them.

## Core Question

```
"What evidence would show [X] is FALSE?"
```

## Procedure

### 1. State the Belief

```
BELIEF: "[clear statement]"
```

### 2. Check Falsifiability

```
"What specific result would prove this FALSE?"

ANSWER: ___________________________

If you cannot answer → belief is NOT FALSIFIABLE → rephrase or reject
```

### 3. Design Falsification Attempt

```
WAYS TO DISPROVE:
1. "If I find [X], belief is false"
2. "If [Y] is true, belief is false"
3. "If expert says [Z], belief is false"

EASIEST TEST: ___
```

### 4. Execute Test

```
ATTEMPT: [what you tried]

RESULT:
[ ] FALSIFIED
    Evidence: ___
    → REJECT or MODIFY belief

[ ] NOT FALSIFIED (tried hard)
    Attempts: ___
    → STRENGTHEN confidence

[ ] CANNOT TEST NOW
    Why: ___
    → FLAG as unverified assumption
```

### 5. Refine if Needed

```
If FALSIFIED but partially true:

ORIGINAL: "[broad claim]"
REFINED:  "[claim] ONLY WHEN [conditions]"
```

## Output

```
BELIEFS TESTED: [count]

┌─────────────────────┬────────────┬─────────────────┐
│ Belief              │ Result     │ Action          │
├─────────────────────┼────────────┼─────────────────┤
│ [belief 1]          │ FALSIFIED  │ REJECT          │
│ [belief 2]          │ SURVIVED   │ STRENGTHEN      │
│ [belief 3]          │ UNTESTABLE │ FLAG            │
└─────────────────────┴────────────┴─────────────────┘

IMPLICATIONS: [what changes]
```
