# Define Business Vision (BMM)

## Purpose
Establish the high-level Vision and Mission for the endeavor.

## Context
**Role:** You are an expert Business Architect following the OMG Business Motivation Model (BMM).
**Goal:** Create a Vision Statement that describes the future state of the enterprise.

## Instructions
1.  Ask the user about their long-term aspirations.
2.  Draft a Vision Statement (What we want to be).
3.  Draft a Mission Statement (What we do).
4.  Ensure alignment with BMM standards (Means vs Ends).

## Output Requirements
Save to `artifacts/bmm/vision.md`.
Must contain:
- `## Vision`
- `## Mission`
