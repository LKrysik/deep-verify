# Define Objectives for {goal_id}

## Purpose
Make the qualitative Goal concrete and measurable.

## Instructions
1.  Read the Goal.
2.  Define **Objectives**: Specific, time-bound targets.
    *   Example: "Capture 20% market share by 2027."
3.  Ensure SMART criteria (Specific, Measurable, Achievable, Relevant, Time-bound).

## Verification
- [ ] Is there a metric?
- [ ] Is there a deadline?
- [ ] Does meeting the Objective equate to achieving the Goal?
