# Export Agents Configuration

## Purpose
Convert abstract Agent definitions into concrete configuration files for the User's CLI (Claude, Gemini, etc.).

## Instructions
1.  Read all Agent definitions from `artifacts/agents/`.
2.  **For Claude CLI:**
    *   Generate a JSON file for each agent in `.claude/commands/`.
    *   Format: `{"name": "{id}", "description": "{description}", "prompt": "{base_prompt}"}`.
3.  **For Gemini CLI:**
    *   Generate appropriate config/profile files (mockup).

## Interaction
*   Ask the user which CLI they are using primarily.
*   Confirm paths before writing.

- [ ] Are the system prompts correctly copied?
- [ ] Are the tools listed?
